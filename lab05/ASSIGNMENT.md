# ЛР5. Ліниві обчислення: генератори та itertools

**Цілі:** generator expressions, `yield`, робота з безкінечними послідовностями.  
**Завдання:** нескінченний стрім Фібоначчі, `sliding_window`, `take`/`drop`/`accumulate`‑практики.

## Необхідно реалізувати
- `fib_stream()` — нескінченний Фібоначчі (islice для обрізання).
- `take(n, it)` / `drop(n, it)` — робота з ітераторами (без матеріалізації).
- `sliding_window(it, k)` — ковзні вікна без створення списків.
- `moving_average(it, k)` — приклад на вікнах.
- Мінімальні обгортки/використання: `accumulate`, `takewhile`, `dropwhile`.

## Що здати
- Код `streams.py` + короткий `main.py` із прикладом запуску.
- Тести `tests/` (pytest).
