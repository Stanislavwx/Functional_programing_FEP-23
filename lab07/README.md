# Лабораторна 7: `functools` та `operator`

Невелика вправа на три ключові теми з функціонального стилю Python:

1. **Конвеєр обробки даних.** Ми беремо сирі рядки про студентів (`RAW_ROWS`), послідовно проганяємо їх через `pipeline`: `mapf(normalize_row)` чистить дані, `filt(adults)` лишає тільки повнолітніх, далі сортування за балом і повернення top‑N. `reduce`, `partial`, `itemgetter` та `methodcaller` дозволяють зібрати цей конвеєр з дрібних функцій без зайвого коду.
2. **Поліморфне завантаження (`singledispatch`).** Функція `load_users` вміє приймати різні типи джерела (рядок, шлях, відкритий файл, вже готовий список). `@singledispatch` автоматично обирає реалізацію залежно від типу аргументу, тому API виглядає єдиним, а логіка для JSON/CSV рознесена по невеликих функціях.
3. **Порівняння об’єктів (`@total_ordering`).** Клас `Box` визначає лише `__eq__` і `__lt__`, а декоратор дописує інші оператори. Порядок побудовано на комбінації об’єму, площі фронтальної грані та ширини, тож коробки можна правильно сортувати й порівнювати.

### Як запускати

```bash
python lab07_funtools_operator.py
```

Скрипт покаже демо конвеєра і прогонить прості тести для всіх трьох блоків.
