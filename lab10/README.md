# Лабораторна 10 — ETL

Це невеликий приклад ETL-пайплайна, де «чисте» ядро (`lab10_etl_core.py`) нормалізує та валідує записи (імʼя, вік, країна, email), а шар I/O (`lab10_etl_io.py`) читає/пише CSV та JSON. У результаті формується набір полів `name`, `age`, `email`, `segment` (adult/minor).

## Як запустити
- `python lab10_etl_core.py` — швидке самотестування ядра.
- `python lab10_tests.py` — повний набір тестів, включно з I/O.
- `python lab10_etl_io.py` — демо-конвертація `lab10_sample.csv` → `lab10_out.json`.

Після запуску тестів або демо результати зберігаються у корені проєкту (`lab10_out.csv`, `lab10_out.json`). Побудовані артефакти можна видалити вручну за потреби.
